Inventory Management Action Screens

Page 03a (View Inventory): Full-screen inventory table with search functionality. Shows all items with SKU, name, quantity, unit price, total value, category, and edit action. Includes comprehensive footer stats.

Page 03b (Add New Item): Form to create new inventory items with fields for SKU, item name, description, category, initial quantity, unit price, and reorder level. Validates unique SKU and required fields.

Page 03c (Update Item): Side-by-side view showing new value (left) and old value (grey, right) for quantity and price updates. Includes change summary showing delta and new totals before saving.
